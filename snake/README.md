# canvas-fn - Snake

 Snake on 

## Usage
Build and serve the Clojurescript files with:
lein figwheel

Open a browser to the default page http://localhost:3449

Run tests in separate tab:
lein figwheel dev test

Generate production files:
lein cljsbuild once release
